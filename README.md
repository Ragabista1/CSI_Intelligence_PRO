# 🚀 CX Intelligence Engine (CSI)

A data-driven Customer Experience (CX) analytics platform designed to transform raw survey data into actionable insights, coaching priorities, and performance intelligence.

## 🎯 Objective

Most dashboards show what happened.  
This platform explains **why it happened** and **what to do next**.

## 🔍 Key Features

- 📊 NPS, CSAT, FCR performance tracking  
- 🧠 Intelligent priority scoring (Risk-based coaching)  
- 👥 Hierarchical drill-down (Manager → Supervisor → TL → Agent)  
- ⚠️ Risk detection engine (Critical / Warning / Healthy)  
- 🎯 Automated coaching recommendations  
- 🧩 Segmentation analysis (Gender, WFH vs Office)  
- 📈 Realistic demo mode with anonymized data  

## 🧠 Intelligence Layer

The system converts performance metrics into:

- Risk scores (Priority Points)  
- Identified skill gaps (Soft Skills, Communication, Problem Solving)  
- Recommended actions (Coaching, Training, Workshops)  

## 🛠️ Tech Stack

- HTML / CSS / JavaScript  
- Chart.js  
- CSV / Excel data integration  

## 🔐 Demo Mode

Includes a fully anonymized dataset with:
- Realistic April performance simulation  
- Structured hierarchy (HC)  
- No real customer data
